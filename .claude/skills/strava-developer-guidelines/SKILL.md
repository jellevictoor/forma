# Strava API Brand Guidelines Skill

This skill encodes Strava's official brand and API usage guidelines (last revised September 29, 2025).
Always apply these rules when helping users build or review Strava-integrated applications.

---

## 1. Connect with Strava — OAuth Button

- **Use the official button** for any OAuth flow; link it to:
  - `https://www.strava.com/oauth/authorize` (web)
  - `https://www.strava.com/oauth/mobile/authorize` (mobile)
- **Download:** `/downloads/1.1-Connect-with-Strava-Buttons.zip`
  - 2 color options: **orange** and **white**
  - Formats: EPS, SVG, PNG
  - Sizes: 48px @1x · 96px @2x

---

## 2. Strava API Logos ("Powered by Strava" / "Compatible with Strava")

- **Download:** `/downloads/1.2-Strava-API-Logos.zip`
  - 3 color options: orange, white, black
  - Formats: EPS, SVG, PNG
  - Variants: horizontal and stacked

### Logo Rules (non-negotiable)
| Rule | Detail |
|------|--------|
| ❌ Never imply Strava sponsorship/development | Logo placement must not suggest official affiliation |
| ❌ Never use as your app's icon | Strava logos cannot be repurposed as app icons |
| ❌ Never modify, alter, or animate | Use logos exactly as provided |
| ✅ Can appear near your logo | Must be completely separate and must **not** be more prominent than your app's name/logo |

---

## 3. Linking Back to Strava Data

When linking to original Strava data sources from within your app:

- **Required link text:** `"View on Strava"`
- Link must be **legible** and **identifiable as a link** — use one of:
  - Bold weight
  - Underline
  - Orange color `#FC5200`

---

## 4. Using the Strava Name & Trademark

### Permitted
- Truthful, factual plain-text references to Strava in feature descriptions
- Phrases: **"Powered by Strava"** or **"Compatible with Strava"** (when referencing interoperability)
- Company name reference: always write **"Strava"** (not "STRAVA", "strava", etc.)

### Prohibited
- Using "Strava" **in your application's name**
- Suggesting your app is **official**, **endorsed**, or **sponsored** by Strava
- Making the Strava name appear in a **larger font** than surrounding text
- Making Strava appear **more prominently** than your app's own name

---

## 5. Compliance Checklist

Use this when reviewing a Strava-integrated app:

- [ ] OAuth button links to the correct Strava URL
- [ ] Official "Connect with Strava" button assets used (not custom-made)
- [ ] "Powered by Strava" / "Compatible with Strava" logo downloaded from official source
- [ ] Logo is separate from app logo, not used as app icon, not modified
- [ ] Back-links to Strava data use "View on Strava" text with correct styling
- [ ] App name does **not** contain "Strava"
- [ ] No language implies official Strava endorsement
- [ ] Strava name font is not larger/more prominent than app name
